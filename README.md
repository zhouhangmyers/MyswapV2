🔧 开发说明
开发框架：使用 Foundry

Solidity 版本：^0.8.26

🛠️ 克隆与初始化项目
# 1. 克隆本项目
git clone https://github.com/zhouhangmyers/MyswapV2.git

# 2. 切换到项目目录
cd MyswapV2

# 3. 在MyswapV2根目录下创建.env文件
添加**DEPLOYER_ADDRESS=你的钱包地址**并保存

# 4. 初始化并安装依赖
make
